# Investigations

These are exploratory notes, not normative contributor guidance.

They document work that informed implementation choices or future options, but
they are not part of the main onboarding path.

## Current Investigations

- [Terminal Column-Width Awareness](column_width_awareness.md) evaluates how to
  make human-readable CLI output width-aware.
- [templ Migration](templ_migration.md) records why no template migration is
  currently needed.

## How To Use This Folder

- treat these documents as background, not policy
- prefer `getting-started/`, `architecture/`, and `reference/` for current guidance
- promote an investigation into `decisions/` only after the project actually
  adopts it
